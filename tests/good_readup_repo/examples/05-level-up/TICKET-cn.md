# 梳理收获并选定下一步

## 1. 目标

把这门课学到的东西收成一条清晰的线, 确认自己达标, 并为接下来的进阶选好一个方向.

---

## 2. 要做的事情

1. 读完这个 mini task 的 README, 对照它复盘整条协作闭环.
2. 不看教程, 用自己的话把 repository, commit, branch, Pull Request, merge 各自解决的问题讲一遍.
3. 从 README 给的进阶方向里挑一个你最感兴趣的, 记下它的搜索关键字和那句话.

**预计用时:** 15 到 30 分钟

---

## 3. 检查清单

- [ ] **能复述闭环**: 能不看教程, 从建 repo 一路讲到 merge, 说清每一步在干什么.
- [ ] **概念过关**: 能用自己的话解释五个核心概念各自解决什么问题.
- [ ] **选定下一步**: 已经挑好一个进阶方向, 并存下了它的关键字和话题.

# 讲得出这段经历

## 1. 目标

能把你怎么学会这条协作闭环的故事讲清楚, 并接得住常见追问, 为对外展示和面试做准备.

## 2. 要做的事情

1. 把这个 mini task 的故事底稿 (README) 从头到尾读一遍, 对默认七幕主线和常见追问心里有数.
2. 用 showcase-demo 排练: 先按主线讲一遍, 再练怎么按不同听众裁剪, 以及怎么接住常见追问.

**预计用时:** 30 到 60 分钟

## 3. 检查清单

- [ ] **读完底稿**: 整份故事底稿过了一遍, 七幕主线和常见追问都清楚.
- [ ] **能流畅讲完**: 用 showcase-demo 排练过, 能不看稿把主线讲顺.
- [ ] **接得住追问**: 常见追问 (为什么学它, 是不是真懂, 遇到什么坑) 都能有理有据地答.

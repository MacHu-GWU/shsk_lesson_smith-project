# GitHub 基础协作课程总览

这是一个 upskill 类型的教学仓库, 教你 GitHub 协作的四个基本功. 所有课程都在 [examples/](examples/) 目录下, 按编号顺序学习:

1. 创建 Repository
2. 创建与编辑文件
3. 使用 Git Branch
4. 合并 Branch

每个 mini task 目录里有 README (教程) 和 TICKET (验收清单). 从 [examples/README-cn.md](examples/README-cn.md) 的课程索引开始.

# 完成 GitHub 基础协作全部课程

## 1. 目标

走完本课程的 4 个 mini task, 建立对 GitHub 协作最小闭环的完整体感: 建仓库, 改文件, 开分支, 合并回主干.

---

## 2. 要做的事情

1. 按顺序完成 examples 下的 4 个 mini task.
2. 每完成一个 task, 对照它自己的 TICKET 检查清单逐项确认.
3. 全部完成后, 从零再独立走一遍完整流程.

**预计用时:** 60 到 90 分钟

---

## 3. 检查清单

- [ ] **四个 task 全部完成**: 每个 task 的 TICKET 检查清单都已逐项打勾.
- [ ] **能独立复现**: 不看教程, 能从建 repo 一路做到 merge branch.
- [ ] **能讲清概念**: 能用自己的话解释 repository, commit, branch, merge 各自解决什么问题.

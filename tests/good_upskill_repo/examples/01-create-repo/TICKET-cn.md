# 创建一个配置合理的 Private Repository

## 1. 目标

独立完成一次 repository 的创建, 并对每个设置项做出有理由的选择.

---

## 2. 要做的事情

1. 在自己的 GitHub 账号下创建一个新 repository, 名字用全小写加连字符.
2. Visibility 选 Private, 打开 Add a README.
3. 按项目类型选择 .gitignore 模板.

**预计用时:** 5 到 10 分钟

---

## 3. 检查清单

- [ ] **命名规范**: repository 名字全小写, 用连字符分隔单词.
- [ ] **Private 可见性**: repo 创建为 Private.
- [ ] **README 就位**: repo 主页能看到 README 内容.
- [ ] **能说清设置**: 能解释 visibility, README, .gitignore 各自的作用.

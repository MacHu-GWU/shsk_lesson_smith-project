# 创建 Branch 并在上面提交改动

## 1. 目标

体验 branch 的隔离效果: 改动只发生在自己的 branch 上, main 保持不变.

---

## 2. 要做的事情

1. 在自己的 repo 里从 main 创建一个新 branch, 名字用小写加连字符.
2. 在新 branch 上编辑文件并至少提交一次.
3. 切回 main, 对比同一个文件在两个 branch 上的内容.

**预计用时:** 10 到 15 分钟

---

## 3. 检查清单

- [ ] **Branch 已创建**: branch 下拉框里能看到新 branch, 命名符合小写连字符规范.
- [ ] **改动已提交**: 新 branch 上有至少一条新的 commit.
- [ ] **Main 未受影响**: 能当场演示同一文件在 main 和新 branch 上内容不同.

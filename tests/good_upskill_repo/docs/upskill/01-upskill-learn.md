# GitHub Collaboration Basics Learning Index

> What there is to learn in this course, where it lives, and in what order to go through it. The upskill-learn skill walks you along this index.

## 1. Study Material

No extra study material outside the guided path. This course is browser-only and hands-on: everything you need to learn lives in the examples progression itself, and each task is something you do on GitHub, not code you read elsewhere. You only need a GitHub account and a web browser. See the [runbook](02-upskill-runbook.md) for the one-time account setup.

## 2. Guided Path

The examples progression is the path the student walks. It breaks into three stretches:

- Set up your repository: examples/01-create-repo to examples/02-edit-files — create a repository and record every change with a commit.
- Branches and Pull Requests: examples/03-git-branch to examples/04-merge-branch — isolate unfinished work on a branch, then merge it back into the main line through a reviewed Pull Request.
- Prove it and level up: examples/05-prove-i-get-it to examples/06-level-up — self-test against the whole course, then pick a next direction.

Each task builds on the repository created in 01, so the student keeps working in the same repo the whole way.

See the [examples theme index](../../examples/README.md) and the [SYLLABUS](../tasks/SYLLABUS.md).

# Syllabus

## 01-create-repo

学完你能独立创建一个配置合理的 GitHub repository, 并说清 visibility, README, .gitignore 各项设置的作用.

## 02-edit-files

学完你能在 GitHub 上创建和编辑文件, 写出一条合格的 commit message, 并看懂文件的修改历史.

## 03-git-branch

学完你能解释 branch 解决什么问题, 并独立创建一个 branch 在上面提交改动而不影响 main.

## 04-merge-branch

学完你能通过 Pull Request 把 branch 合并回 main, 并说清 PR 在团队协作里的审查作用.

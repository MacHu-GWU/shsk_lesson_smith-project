# Syllabus

## 01-create-repo

By the end you can independently create a well-configured GitHub repository and explain what the visibility, README, and .gitignore settings do.

## 02-edit-files

By the end you can create and edit files on GitHub, write a proper commit message, and read the change history of a file.

## 03-git-branch

By the end you can explain what problem branches solve and independently create a branch and commit to it without touching main.

## 04-merge-branch

By the end you can merge a branch back into main through a Pull Request and explain the review role a PR plays in teamwork.

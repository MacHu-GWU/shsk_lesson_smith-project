# Complete All GitHub Basics Lessons

## 1. Goal

Work through the 4 mini tasks of this course to build a complete feel for the minimal GitHub collaboration loop: create a repo, edit files, open a branch, merge back to main.

---

## 2. What to Do

1. Complete the 4 mini tasks under examples in numbered order.
2. After each task, go through its own TICKET checklist item by item.
3. Once done, walk through the whole flow again from scratch on your own.

**Estimated time:** 60 to 90 minutes

---

## 3. Checklist

- [ ] **All four tasks completed**: every item in each task's TICKET checklist is checked.
- [ ] **Independent reproduction**: you can go from creating a repo to merging a branch without the tutorials.
- [ ] **Concepts explained**: you can explain in your own words what repository, commit, branch, and merge each solve.

# GitHub Basics Course Overview

This is an upskill teaching repository covering four fundamental GitHub collaboration skills. All lessons live under [examples/](examples/), to be studied in numbered order:

1. Create a Repository
2. Create and Edit Files
3. Work with Git Branches
4. Merge a Branch

Each mini task directory contains a README (the tutorial) and a TICKET (the acceptance checklist). Start from the course index at [examples/README.md](examples/README.md).

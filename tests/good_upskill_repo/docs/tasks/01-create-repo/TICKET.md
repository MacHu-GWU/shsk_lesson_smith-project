# Create a Well Configured Private Repository

## 1. Goal

Independently create a repository, making a reasoned choice for every setting.

---

## 2. What to Do

1. Create a new repository under your own GitHub account, named in all lowercase with hyphens.
2. Set visibility to Private and turn on Add a README.
3. Pick a .gitignore template matching the project type.

**Estimated time:** 5 to 10 minutes

---

## 3. Checklist

- [ ] **Naming convention**: the repository name is all lowercase with hyphen-separated words.
- [ ] **Private visibility**: the repo is created as Private.
- [ ] **README in place**: the repo homepage shows the README content.
- [ ] **Settings explained**: you can explain what visibility, README, and .gitignore each do.

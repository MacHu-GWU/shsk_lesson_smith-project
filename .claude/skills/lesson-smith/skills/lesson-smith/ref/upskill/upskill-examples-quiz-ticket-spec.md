# 自测通关

## 1. 目标

检验自己是否真的学会了这门课, 不只知其然, 更知其所以然.

## 2. 要做的事情

1. 把这个 mini task 的题库 (README) 从头到尾读一遍: 每道题先自己答, 再对照参考回答与深入解读.
2. 用 /upskill-quiz 抽测自己, 反复测到正确率稳定在 70% 以上.

**预计用时:** 30 到 60 分钟

## 3. 检查清单

- [ ] **读完题库**: 整个题库从头到尾过了一遍, 每题都先自答再对照.
- [ ] **测试通关**: /upskill-quiz 的正确率达到 70% 以上.
